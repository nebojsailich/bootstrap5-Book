Inside the Book

JavaScript lies at the heart of almost every modern web application, from social apps like Twitter to browser-based game frameworks like Phaser and Babylon. Though simple for beginners to pick up and play with, JavaScript is a flexible, complex language that you can use to build full-scale applications. This much anticipated and thoroughly revised third edition of Eloquent JavaScript dives deep into the JavaScript language to show you how to write beautiful, effective code. It has been updated to reflect the current state of JavaScript and web browsers and includes brand-new material on features like class notation, arrow functions, iterators, async functions, template strings, and block scope. A host of new exercises have also been added to test your skills and keep you on track.

Book Reviews

What do you think of the Book ?

Excellent JavaScript book for beginners

This book is unique in that it starts out with JavaScript as a programming language, and continues delving further and further into how the language works - but without touching on the browser model until several chapters in. It comes at JavaScript purely as a programming language first, then getting into its uses (which include browser programming) later.

Review by Marko
Not for absolute begginers

I was required to purchase this book for a JavaScript class. This book is not for absolute beginners or those with minimal programming experience. This book is more tailored to those with programming experience who want to learn JavaScript. If this is you the book is better suited for you. If you are learning JavaScript primarily or solely for web design purposes skip this book.

Review by Nebojsa
Very readable and informative

Most books on programming are quite boring to read. This is a pleasant exception to that rule. I found the book both quite readable and informative. Personally I love JavaScript as a programming language. This book has helped be move back to JavaScript after using some other more rigid languages. NOT done yet, but it is well written. The beginning works well for all programming languages not just JavaScript. I found the example a tad hard to follow, but then again I would rather do it than read about it!

Review by Sara
Not Very Thorough

This book does not really give much information about javascript specifically. I found it very distracting to find errors in the code. There were words that were misspelled and sentences that were poorly formed. Even though this does not speak to the technical part of the publication, it did make reading it annoying. I would consider the book okay but not anything special.

Review by Tamara
An exhaustive introduction and a good refresher.

I definitely recommend this book to someone who has used the language before and could use a refresher. You will learn or remember several interesting features that will improve your work. I also recommend this to a new programmer, but with the caveat that this will not give a complete education on the language. It attempts to and mostly succeeds, but Haverbeke's opinionated style prevents this from being a complete education on best practices. Nevertheless, I don't think you can do much better: someone coming to the language with no experience will always improve in both style and quality, and this book provides an excellent introduction to both.

Review by Nemanja
